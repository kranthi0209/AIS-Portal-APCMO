📌 Important Rules
  1. One row = one post. If an officer holds 2 posts, add 2 rows with the same Identity No.
  2. Each officer must have exactly one Regular post (primary post). FAC posts are additional.
  3. For officers who held no post during this period, enter No Posts Held as the Post Name.
  4. Do not add or remove officer rows from the template. Only fill columns D onwards.
  5. Files Pending is auto-computed: Opening Balance + Created + Received − Processed.
